AD-related SNP-RBP binding

This function tab has 4 significance level inputs for the user to choose from: 0.05, 0.01, 1 x 10-6 and 5 x 10-8. The latter two p-value accounts for multiple hypothesis testing in GWAS study and will return the most significant results.
After clicking on the submit button, 3 subtabs (Description, Table, and Plot) will show up for the user to check out.

Description subtab returns summary statistics of RBP and its binding SNPs to the users.

Table subtab displays the results in a table having 7 columns:

Chromosome: On which chromosome the SNP-RBP interaction occurs.
SNP position : Start position and end position of the SNP on the chromosome.
SNP marker name : The name of the SNP biological marker.
RBP name : The name of RNA Binding Protein.
RBP p_value in (-log10): This indicates the significance of the interaction between SNP and RBP.
The p-value of the SNP: This indicates the significance of the association with Alzheimer's Disease.
RBP target gene: Genes annotated to the SNP position with hyperlinks to the genes' GenBank® result pages.

Plot subtab displays the distribution of SNP counts across RBPs.

Loci Number

This function tab takes the RBP name as the input.
After clicking on the submit button, 3 subtabs (Description, Table, and Plot) will show up for the user to check out.

Description subtab returns summary statistics of the SNPs related to the RBP input and the genes annotated to the SNPs.

Table subtab displays the results in a table having 5 columns:

SNP chromosome: On which chromosome the SNP-RBP interaction occurs.
SNP position: Start position and end position of the SNP on the chromosome.
SNP marker name: The name of the SNP biological marker.
P-value of SNP: Which indicates the significance of the association with Alzheimer's Disease.
Target gene: Genes annotated to the SNP position with hyperlinks to the genes' GenBank® result pages.

Plot subtab displays the location of SNPs on the human chromosomes. The top is the chromosome start and the bottom is the chromosome end. Each triangle marker on the chromosome represents a SNP. Each chromosome can be clicked on to show the enlarged individual view of it.

AD-related RBP function

This function tab takes the SNP name as the input.
After clicking on the submit button, 3 subtabs (Description, Table, and Plot) will show up for the user to check out.

Description subtab returns the beta value of the SNP (provided by Dr. Xiaoling Zhang), which indicates whether the SNP is positively or negatively associated with AD.

Table subtab displays the results in a table having 4 columns:

RBP name: The name of RNA Binding Protein.
RBP Gene: The gene ensemble ID annotated to the RBP binding location.
RBP function: The function of RNA binding protein.
Target: Genes annotated to the RBP position with hyperlinks to the genes' GenBank® result pages.

Plot subtab displays the distribution of RBP counts across RBP functions.
